## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/hackathon-showcase.git
   cd hackathon-showcase
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Setup Database**
   ```bash
   npx prisma db push
   npx prisma db seed
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open http://localhost:3000**
